## 使用方法

计算先验时，把联赛 λ_avg 设为上表对应值。例如分析英超比赛：

```
λ_avg = 2.85
主队基础先验 = (主队主场进球场均) × (1 + 主场加成)
客队基础先验 = (客队客场进球场均)
```

## 常见坑

1. **赛季初数据不可靠**：前 5 轮的场均数据方差大，建议加权用上赛季数据
2. **杯赛差异大**：欧冠淘汰赛和小组赛风格完全不同，不能混用
3. **保级/争冠心态**：赛季末关键战役进球数会偏离均值
4. **冬休期后状态**：德甲、意甲冬歇回来第一轮偏保守
