# 竞彩足球超深度流程分析 Skill

## 适用场景

- 竞彩购买前的结构化赛前分析
- 足球博主/内容创作者的赛事解读稿
- 贝叶斯统计 + 泊松比分预测的教学案例
- 五大联赛、欧冠、欧联、欧会、世界杯预选赛、亚冠、中超、日韩联赛

**重要声明**：本 Skill 仅做数据分析和娱乐性预测，不构成投注建议。竞彩有风险，理性参与，未成年人严禁参与。

## 核心七步流程

### 第一步：基础信息卡

输出结构化基础信息：

```
赛事：X 联赛 第 N 轮
时间：YYYY-MM-DD HH:MM
主队 VS 客队
主场：XX 球场（容量/海拔/气候）
裁判：XX（本赛季红黄牌场均）
天气：晴/雨/雪 + 温度
```

### 第二步：数据采集（不允许跳过）

**必须拿到的数据源**：

| 指标 | 来源 | 用途 |
|---|---|---|
| xG / xGA | FBref.com / Understat | 进攻/防守真实水平 |
| 近 10 场战绩 | Soccerway | 状态 |
| 主客场分开战绩 | Soccerway | 主场优势 |
| 直接交锋 H2H 近 5 次 | Soccerway | 风格相性 |
| 伤停名单 | PhysioRoom / 队官网 | 阵容变化 |
| 预测首发（可能阵型） | WhoScored | 战术布置 |
| 盘口（初盘/即时盘） | 亚盘/欧赔 | 市场定价 |
| 大小球盘 | 立博/必发 | 进球预期 |

**注意**：如果某项数据缺失，**必须明确声明缺失**，不能编造。

### 第三步：贝叶斯先验建立

基于**赛季整体数据**建立先验分布：

```
先验：
  主队主场进球均值 λ₁ = (主场总进球 / 主场场次)
  客队客场进球均值 λ₂ = (客场总进球 / 客场场次)
  联赛平均进球 λ_avg = 2.6（五大联赛常年在 2.5-2.8）
```

### 第四步：后验更新（关键一步）

用本场**对手防守水平**调整先验：

```
后验：
  μ₁ = λ₁ × (客队客场失球率 / 联赛平均失球率) × 伤停系数 × 天气系数
  μ₂ = λ₂ × (主队主场失球率 / 联赛平均失球率) × 伤停系数 × 天气系数
```

**系数表**：
| 因素 | 系数范围 |
|---|---|
| 主力伤停（前锋/中场核心/主力后卫） | 0.85 ~ 0.95 |
| 大雨/大雪 | 0.90（压低进球）|
| 夜间比赛 | 1.00（无影响）|
| 高原客场（海拔>1500m） | 0.92 |
| 密集赛程第 3 场 | 0.93 |

### 第五步：泊松分布比分预测

用 μ₁、μ₂ 生成比分概率矩阵：

```
P(主队进 i 球) = (μ₁^i × e^-μ₁) / i!
P(客队进 j 球) = (μ₂^j × e^-μ₂) / j!
P(比分 i:j) = P(主队 i) × P(客队 j)
```

生成 0:0 到 4:4 的概率矩阵，列出：
- Top 3 最可能比分
- 胜/平/负概率
- 大球（>2.5）概率
- 双方进球概率

### 第六步：战术克制与剧本预测

这一步**超越数据**，进入足球理解：

**战术克制分析**：
- 主队阵型 vs 客队阵型（4-3-3 vs 3-5-2 边路优势？）
- 高位压迫 vs 防守反击
- 定位球攻防（客队本赛季 30% 进球来自定位球，主队丢球 25% 来自定位球 → 关键！）
- 中场控制权（xPass、传球成功率）

**剧本预测**（三种典型）：

| 剧本 | 特征 | 概率 |
|---|---|---|
| 主队高开控盘 | 主队开场 15 分钟内破门 → 控盘拖时间 | 35% |
| 客队偷袭反击 | 客队 1-2 次反击得手 | 25% |
| 闷平局 | 双方互相忌惮，节奏慢 | 20% |
| 开放对攻 | 双方都要赢，互攻互进 | 20% |

### 第七步：结论输出（固定格式）

```markdown
## 最终结论

**推荐关注**（仅做分析参考）：
1. 比分 Top3：X:Y (p=?%) / A:B (p=?%) / C:D (p=?%)
2. 胜平负：胜 ?% / 平 ?% / 负 ?%
3. 大小球：大 ?% / 小 ?%
4. 双方进球：是 ?% / 否 ?%

**剧本**：最可能走 XX 剧本（?%）

**关键变量**：
- 变量 1（如 X 球员能否出场）
- 变量 2（如天气变化）

**风险提示**：
- 数据样本不足（如赛季初数据）
- 历史数据不代表未来
- 竞彩有风险，理性参与
```

## 使用示例

**用户**：
> 用 jingcai-football-deepflow 分析下周六利物浦 vs 阿森纳

**Skill 输出**（摘要）：

**第一步**：基础信息
> 英超第 X 轮，2026-05-18 23:30，安菲尔德，主裁：迈克尔·奥利弗（场均 4.2 黄 + 0.3 红）。天气晴，16°C。

**第二步**：数据
> 利物浦主场：场均 2.3 球，失球 0.9；阿森纳客场：场均 1.6 球，失球 1.1。H2H 近 5 场 2 胜 1 平 2 负，场均 2.8 球。萨拉赫确认首发，萨卡轻伤待定。

**第三步**：先验
> λ₁ = 2.3（主场进球），λ₂ = 1.6（客场进球）

**第四步**：后验
> μ₁ = 2.3 × (1.1/1.4) × 1.00 ≈ **1.81**
> μ₂ = 1.6 × (0.9/1.3) × 0.95（萨卡可能缺席）≈ **1.05**

**第五步**：泊松矩阵
> Top3 比分：1:1 (10.9%) / 1:0 (10.4%) / 2:1 (9.9%)
> 胜平负：利物浦 54.9% / 平 23.0% / 阿森纳 21.8%
> 大小球：大 2.5 球 54.2%

**第六步**：剧本
> 主队控盘 40%（利物浦主场近 8 场 6 胜）
> 对攻 30%（双方都需要积分）

**第七步**：结论
> 推荐分析方向：让球盘 -0.5（利物浦胜），大小球倾向大 2.5。关键变量：萨卡伤情、迪亚斯复出时间。

## 关键原则

1. **拒绝"我觉得"**：每个结论都要给数据来源
2. **承认数据缺失**：拿不到就写"数据缺失"，不编
3. **贝叶斯不是万能的**：样本 < 5 场时置信度低，必须标注
4. **合规表达**：用"推荐关注"、"分析方向"，不用"推荐投注"、"稳赢"
5. **防止赌博推广**：每次输出必须带风险提示

- 杯赛单场决赛（样本太少，先验不可靠）
- 赛季首轮（历史数据断层）
- 涉赌场合（拒绝输出"稳赢"、"必胜"等表达）
